### Step1. 安装docker

* [下载链接](https://docs.docker.com/get-docker/)

### Step2. 下载部署工程

```
git clone https://github.com/archguard/archguard_deploy.git 
```

### Step3. 执行命令
- Start all containers
`./deploy.sh up`
- Stop all containers
`./deploy.sh stop`
- Remove all containers
`./deploy.sh rm`
