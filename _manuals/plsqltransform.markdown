# 改造工具
## PL/SQL 转 Kotlin工具
源自：ThoughtWorks

状态：已集成ArchGuard

工具适用场景：
- 能解决80%的PL/SQL语法转kotlin

步骤：

- 安装ArchGuard: [https://archguard.org/operationManual](https://archguard.org/operationManual)
- 点击"PL/SQL转Kotlin"，进入工具
- 输入包名，将PL/SQL复制到左边的输入框
- 点击"转换"，在右边的输入框查看结果
